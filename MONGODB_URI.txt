MongoDB Connection URI for RideShare Local Project

===========================================
LOCAL MONGODB (Current Configuration)
===========================================

mongodb://localhost:27017/rideshare

Use this in MongoDB Compass:
1. Open the "New Connection" modal
2. Paste: mongodb://localhost:27017/rideshare
3. Name it: "RideShare Local"
4. Click "Save & Connect"

===========================================
IF USING MONGODB ATLAS (Cloud)
===========================================

mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/rideshare?retryWrites=true&w=majority

Replace:
- username: Your Atlas database username
- password: Your Atlas database password  
- cluster0.xxxxx: Your actual cluster address
